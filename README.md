# ERC721

This is a standard. 
Why ERC-721?

-ERC-721 = a unique ID card
-One person owns it
-It proves who you are
-You either have it or you don’t

Why choose ERC-721A over ERC-721
-Much lower gas fees
Minting multiple NFTs in ERC-721 is expensive because each mint costs gas.
ERC-721A reduces this cost significantly.
Batch minting (one transaction)
ERC-721A lets you mint many NFTs in a single transaction instead of repeating the process.
Better user experience
Users pay less and complete minting faster, especially during launches.
Ideal for large collections
If you’re creating hundreds or thousands of NFTs, ERC-721A saves a lot of money.
Still fully compatible
NFTs are still unique and behave like standard ERC-721 tokens.
